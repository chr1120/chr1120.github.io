🎯 프로젝트 핵심 가치

PPE Facility Management System은 가스 측정기와 분석기를 전문으로 하는 전자상거래 플랫폼입니다. 구매자와 판매자를 연결하는 B2B/B2C 통합 시스템으로, 화학물질 기반의 정확한 제품 분류 및 검색 시스템을 제공하며, 현금 및 포인트 결제 시스템을 지원합니다.

🚀 핵심 기능

🔍

화학물질 기반 검색

화학물질 상태(기체/액체/고체)에 따른 동적 분류와 실시간 API 연동으로 정확한 제품 검색을 제공합니다.

🛒

스마트 장바구니

세션 기반 장바구니 관리로 실시간 수량 조정, AJAX 기반 비동기 업데이트를 지원합니다.

💳

이중 결제 시스템

현금과 포인트 결제를 모두 지원하며, 포인트 할인율 계산 시스템이 내장되어 있습니다.

📦

완전한 주문 관리

주문 생성부터 배송까지 전체 주문 생명주기를 관리하고 상태별 추적이 가능합니다.

⚙️

관리자 시스템

통합 대시보드, 상품/주문/포인트 관리, 검색/필터링, 페이징 시스템을 제공합니다.

💬

💡 핵심 구현 기술

🔬 화학물질 기반 제품 분류 시스템

화학물질과 제품의 정확한 매칭을 위한 고급 분류 시스템으로, 다대다 관계의 복잡한 쿼리를 최적화했습니다.

-- 화학물질 기반 제품 검색의 핵심 쿼리 SELECT p.* FROM product p JOIN product_chemical pc ON p.seq = pc.product_seq JOIN chemical c ON pc.chemical_id = c.id WHERE (:productType IS NULL OR p.product_type = :productType) AND c.name IN (:chemicalNames) AND (:states IS NULL OR c.state IN (:states)) GROUP BY p.seq HAVING COUNT(DISTINCT c.name) = :chemicalCount

🎯 지원하는 필터링 옵션

  • 화학물질명: 다중 선택 가능한 화학물질 기반 검색
  • 화학물질 상태: 기체, 액체/고체 상태별 필터링
  • 제품 유형: 측정기, 분석기 등 제품 카테고리
  • 용도: 산업용, 실험실용 등 사용 목적
  • 측정 방식: 단일/다중 측정 방식
  • 키워드 검색: 제품명, 모델명 등 텍스트 검색
💰 포인트 변환 시스템

현금과 포인트 간의 정확한 환율 계산과 할인율 적용 로직을 구현했습니다.

@Component public class PointConverter { private static final double POINT_TO_WON_RATIO = 1000.0; // 1포인트 = 1,000원 public static int pointsToWon(int points) { return (int) (points * POINT_TO_WON_RATIO); } public static int wonToPoints(int won) { return (int) Math.ceil(won / POINT_TO_WON_RATIO); } public static int calculateDiscountPoints(int totalAmount) { if (totalAmount >= 200000) { return 230; // 200,000원 이상 시 230포인트 할인 } else if (totalAmount >= 100000) { return 100; // 100,000원 이상 시 100포인트 할인 } return 0; } }

💳 결제 시스템 특징

  • 이중 결제: 현금과 포인트 동시 지원
  • 자동 할인: 구매 금액에 따른 자동 포인트 할인
  • 정확한 환율: 1포인트 = 1,000원 고정 환율
  • 할인 등급: 10만원/20만원 이상 시 차등 할인

@Service @Transactional public class OrderService { public Order createCashOrder(String sessionId, String buyerName, String buyerEmail, String buyerPhone, String shippingAddress) { // 1. 장바구니 유효성 검증 List<CartItem> cartItems = cartService.getCartItems(sessionId); if (cartItems.isEmpty()) { throw new IllegalStateException("장바구니가 비어있습니다."); } // 2. 주문 생성 및 주문 아이템 설정 Order order = Order.builder() .orderNumber(generateOrderNumber()) .sessionId(sessionId) .buyerName(buyerName) .buyerEmail(buyerEmail) .buyerPhone(buyerPhone) .shippingAddress(shippingAddress) .totalAmount(calculateTotalAmount(cartItems)) .paymentMethod(PaymentMethod.CASH) .orderStatus(OrderStatus.PENDING) .build(); return orderRepository.save(order); } }

🗄️ 핵심 테이블 구조

📊 데이터베이스 스키마

제품 관련 테이블:

  • product: 제품 기본 정보 (카테고리, 유형, 가격, 이미지 등)
  • chemical: 화학물질 정보 (이름, 알파벳, 상태, 분자식 등)
  • product_chemical: 제품-화학물질 관계 테이블 (다대다 관계)

거래 관련 테이블:

  • cart: 장바구니 정보 (세션별 관리)
  • cart_item: 장바구니 아이템 (제품, 수량, 가격)
  • orders: 주문 정보 (주문번호, 구매자 정보, 결제 방법)
  • order_items: 주문 아이템 (제품, 수량, 단가, 총액)
  • user_points: 사용자 포인트 정보 (세션별 포인트 관리)

커뮤니케이션 테이블:

  • qna: 제품 문의 (질문, 답변, 상태)
  • review: 제품 리뷰 (평점, 제목, 내용, 이미지)
  • quote_request: 견적 요청 (회사 정보, 수량, 메시지)
  • calibration_request: 검교정 요청 (장비 정보, 요청 내용)
🏗️ 주문 테이블 스키마
-- 주문 테이블 CREATE TABLE orders ( id BIGINT PRIMARY KEY AUTO_INCREMENT, order_number VARCHAR(50) UNIQUE NOT NULL, session_id VARCHAR(255) NOT NULL, buyer_name VARCHAR(100) NOT NULL, buyer_email VARCHAR(200) NOT NULL, buyer_phone VARCHAR(20) NOT NULL, shipping_address TEXT NOT NULL, total_amount INT NOT NULL, payment_method ENUM('CASH', 'POINTS') NOT NULL, points_used INT, order_status ENUM('PENDING', 'CONFIRMED', 'PROCESSING', 'SHIPPED', 'DELIVERED', 'CANCELLED', 'REFUNDED') NOT NULL, is_deleted BOOLEAN DEFAULT FALSE, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP ); -- 사용자 포인트 테이블 CREATE TABLE user_points ( id BIGINT PRIMARY KEY AUTO_INCREMENT, session_id VARCHAR(255) UNIQUE NOT NULL, total_points INT DEFAULT 0, used_points INT DEFAULT 0, available_points INT DEFAULT 0, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP );
